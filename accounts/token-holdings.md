# Token Holdings

```
The Token Holdings tab displays all ERC-compliant assets held by an address.
```

### Supported Standards

Supported token standards:

• TC-20

• ERC-20\
• ERC-721 (NFT)\
• ERC-1155

### ERC-20 Section

```
## ERC-20 Tokens
```

Displays:

• Token name\
• Symbol\
• Contract address\
• Token balance\
• Transfer history

### NFT Section

```
## NFTs
```

NFT pages include:

• Token ID\
• Metadata\
• Image\
• Ownership history

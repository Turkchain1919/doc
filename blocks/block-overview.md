# Block Overview

```
A block represents a batch of validated transactions permanently recorded on the Turkchain ledger.
```

```
## Block Details
```

Each block page on TurkScan includes:

• Block Number\
• Timestamp\
• Validator\
• Transactions Count\
• Gas Used\
• Gas Limit\
• Block Size\
• Parent Block\
• State Root\
• Transactions Root\
• Receipts Root\
• Hash

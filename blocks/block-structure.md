# Block Structure

```
Each block consists of a header and a body.
```

### Block Header

```
## Block Header
```

```
The block header contains cryptographic and consensus-critical metadata:

• Parent Hash  
• State Root  
• Transactions Root  
• Receipts Root  
• Timestamp  
• Block Number  
• Gas Limit  
• Gas Used  
• Validator Signature   
```

### Block Body

```
## Block Body
```

The block body contains:

• List of transactions\
• Transaction receipts\
• Event logs

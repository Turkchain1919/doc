# MetaMask issues

```
MetaMask issues are typically related to network configuration.
```

### Common Issues

```
• Wrong network selected  
• Incorrect chain ID  
• Gas estimation failure  
• Stuck transaction  
• Insufficient funds for gas 
```

### Fix Stuck Transaction

```
1. Go to MetaMask settings  
2. Enable "Customize transaction nonce"  
3. Send replacement transaction with same nonce and higher gas price 
```

### Reset Account (Advanced)

```
Settings → Advanced → Reset Account

This clears local transaction history (does not remove funds).
```

# Connect to Turkchain

Turkchain is accessible through its public RPC endpoint.

RPC URL:\
https://rpc.turkchain1919.com

Chain ID:\
1919

Currency Symbol:\
TURK

Decimals:\
18

Explorer: https://turkscan.com

JSON-RPC Example

```
// {
  "jsonrpc":"2.0",
  "method":"eth_blockNumber",
  "params":[],
  "id":1
}
```

```
This request returns the latest block number from the Turkchain network.
```

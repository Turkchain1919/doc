# Using TurkScan Explorer

```
TurkScan provides full transparency into the Turkchain network.
```

### Search Capabilities

You can search by:

* Transaction Hash
* Block Number
* Wallet Address
* Token Contract

### Address Page Breakdown

Address pages include:

* TURK balance
* Token balances
* Transaction history
* Internal transactions
* Contract tab (if applicable)

### Transaction Page Breakdown

```
Transaction details include:

- Status
- Block number
- Gas used
- Gas price
- From / To
- Input data
- Logs
```

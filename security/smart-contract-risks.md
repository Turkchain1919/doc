# Smart Contract Risks

```
Smart contracts are immutable once deployed unless upgradeable patterns are used.
```

### Common Vulnerabilities

```
• Reentrancy  
• Integer overflow / underflow  
• Access control issues  
• Unchecked delegatecall  
• Flash loan manipulation  
```

### Audit Importance

```
All production contracts should undergo independent security audits.
```

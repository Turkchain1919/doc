# Validator Security

```
Validators are responsible for maintaining network integrity.
```

### Responsibilities

```
• Block proposal  
• Transaction validation  
• Signature verification  
• State transition execution  
```

### Slashing & Misbehavior

```
Malicious or faulty validators may face penalties depending on network governance rules.
```

### Operational Best Practices

```
• Secure private keys  
• Use hardware security modules  
• Monitor uptime  
• Maintain node redundancy  
```

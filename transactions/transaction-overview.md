# Transaction Overview

```
A transaction represents a signed instruction sent from an externally owned account (EOA) to the Turkchain network.
```

### Transaction Lifecycle

```
## Transaction Lifecycle
```

Transaction flow:

Wallet → RPC Node → Mempool → Validator Inclusion → Block Confirmation → Finality

### Transaction Page Fields

```
## Transaction Details
```

Each transaction page on TurkScan includes:

• Transaction Hash\
• Status\
• Block Number\
• Timestamp\
• From\
• To\
• Value\
• Gas Limit\
• Gas Used\
• Gas Price\
• Transaction Fee\
• Nonce\
• Input Data\
• Logs
